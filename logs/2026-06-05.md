## Bugs
- **FIX-001** — PDF ArrayBuffer detachado pelo worker do PDF.js v4. Causa: `postMessage` com `transfer`. Solução: Blob URL. Arquivos: AppContext.jsx + PdfViewer.jsx.
- **FIX-002** — CSV edição com filtro ativo descartava linhas fora do filtro. Causa: `commitEdit` reconstruía CSV a partir do `filtered`. Solução: `__dataIdx` + operar sobre `data` original. Arquivo: CsvViewer.jsx.

## Aprendizados / armadilhas
- PDF.js v4 transfere o ArrayBuffer para o worker — nunca armazenar ArrayBuffer de PDF em estado React. Usar Blob URL sempre. → Armadilha #1 CONTEXT.md.
- `onBlur` sempre dispara antes de `onClick` de outro elemento. Para botão de cancelar que compete com `onBlur`, usar `onMouseDown + preventDefault`. → Armadilha #5 CONTEXT.md.
- Ao editar dados com filter/sort aplicado, nunca reconstruir dados persistentes a partir do array derivado — manter referência ao índice original. → Armadilha #2 CONTEXT.md + FIX-002.
